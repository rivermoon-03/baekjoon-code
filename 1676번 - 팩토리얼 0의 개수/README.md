# 1676번: 팩토리얼 0의 개수

## 문제 설명


<p>N!에서 뒤에서부터 처음 0이 아닌 숫자가 나올 때까지 0의 개수를 구하는 프로그램을 작성하시오.</p>



## 입력 형식


<p>첫째 줄에 N이 주어진다. (0 ≤ N ≤ 500)</p>



## 출력 형식


<p>첫째 줄에 구한 0의 개수를 출력한다.</p>



## 예제

### 예제 입력 1

```
10

```

### 예제 출력 1

```
2

```
          

### 예제 입력 2

```
3

```

### 예제 출력 2

```
0

```
